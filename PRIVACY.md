# Privacy Policy for JSON Side

**Last Updated: June 2024**

## Data Collection

JSON Side does **NOT** collect, store, or transmit any user data to external servers.

## Local Storage

All data is stored locally in your browser:
- **License key**: Stored in browser bookmarks (only you can access)
- **History records**: Stored in browser local storage
- **Settings**: Stored in browser sync storage

## Third-Party Services

JSON Side operates completely offline. No data is sent to:
- No analytics services
- No advertising networks
- No third-party APIs

## Permissions Used

| Permission | Purpose |
|------------|---------|
| `contextMenus` | Add right-click menu for "Format JSON" |
| `storage` | Save settings and history locally |
| `bookmarks` | Store license key securely |

## Your Rights

- All your data stays on your device
- You can delete all data by removing the extension
- No account registration required
